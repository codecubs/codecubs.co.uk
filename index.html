CodeCubs

CodeCubs educates children in all aspects of technology. Technology moves faster every day. Will our future generation keep up? Our courses are designed to engage students and prepare them for the future of technology.

At CodeCubs, students will learn to code by creating programs, websites and video games! Each course is flexible, ensuring that children learn what they won't from school.

Courses

01. Bash Basics

Every computer at CodeCubs runs Linux. Linux is a coder-friendly operating system that reveals everything under-the-hood. Students will learn through Linux core concepts of operating systems - such as how to program them!

Lessons are kept engaging with fun commands, trivia and pop-quizzes

02. JavaScript Programming

JavaScript is the most popular web-based programming language. Every website you visit - including this one - runs with JavaScript! This course will serve as an introduction to JavaScript. Introducing key concepts behind any programming language.

Knowledge from Bash Basics will be built upon in this course, by creating real websites and games.

03. NodeJS Programming

We take JavaScript even further with NodeJS. Node allows us to code our computer directly, outside of a web browser. Students will use Node to create more advanced websites and games.

This course puts students to the test and allows them to think about websites in a whole new way!
